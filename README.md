# Abode Assessment

## Getting Started

Run development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Required .env Variables
```
OPENAI_API_KEY
```

## Stack

- Next.js
- Tailwind CSS
- Shadcn UI
- OpenAI
- React
- TypeScript
- ESLint
- Prettier
- pnpm

## Features

- Generate a target name from a prompt
- Verify candidate names against the latest target
- Run the provided test suite to see pass/fail (names from the test cases provided)
